# # PEP: Python Enhancement Proposals
* [PEP 0 – Index of Python Enhancement Proposals (PEPs)](https://peps.python.org/pep-0000/)
* [PEP 1 – PEP Purpose and Guidelines](https://peps.python.org/pep-0001)

PEP Types Key:
* I: Informational
* P: Process
* S: Standards Track

PEP Status Key:
* A: Accepted
* A: Active
* D: Deferred
* `<No letter>`: Draft
* F: Final
* P: Provisional
* R: Rejected
* S: Superseded
* W: Withdraw
